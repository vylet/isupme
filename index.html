Welcome to the "Down For Me?" extension page.

Thanks for installing this addon and you can report issues or request features here.

Changes

You can now set a timeout period in the extensions options. You can access the options in addons page. This extension will then check for the status of the page if the page doesn't respond in the timeout period. If the timeout is less than 0 will make the extension check only when there is an error page.

For other browsers

This webextension can be easile be ported to chrome , opera and other chromium based browsers. If you want it for a specific browser post your request here.
